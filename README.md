# Air Quality Data Analysis (AI)

## Overview
This project analyzes air quality data using AWS PartyRock's AI-powered data analysis tool. The goal is to extract meaningful insights, identify trends, and understand pollution patterns over time.

## Dataset
The dataset contains air quality measurements from a monitoring station, including:
- PM10 (coarse particulate matter)
- PM2.5 (fine particulate matter)
- Carbon Monoxide (CO)
- Nitrogen Dioxide (NO₂)
- Ozone (O₃)
- Sulfur Dioxide (SO₂)

## Analysis Performed
- Identified key pollutants and their impact
- Analyzed trends over time
- Examined daily variation patterns
- Compared pollutant levels
- Generated summary statistics table

## Key Insights
- PM10 and PM2.5 are the dominant pollutants with high variability
- Air quality shows significant spikes during certain periods
- Daily patterns indicate higher pollution in the afternoon and early morning
- Gaseous pollutants remain relatively low compared to particulate matter

## Tools Used
- AWS PartyRock (Analyze Data)
- Generative AI for data interpretation
